# speedtest

测速用例 - 生成指定大小的随机文件，用于磁盘/网络速度测试。

## 构建

```bash
go build -o gen .
```

## 用法

```bash
./gen <大小MB>
```

- 参数为正整数，单位为 MB
- 生成文件名为 `<大小>mb.txt`
- 文件内容为随机乱序的英文字母、数字、标点符号

## 示例

```bash
./gen 100    # 生成 100mb.txt (100 MB)
./gen 1      # 生成 1mb.txt (1 MB)
./gen 1024   # 生成 1024mb.txt (1 GB)
```

## 生成内容字符集

- 小写字母: `a-z`
- 大写字母: `A-Z`
- 数字: `0-9`
- 标点符号: `!@#$%^&*()-_=+[]{}|;:',.<>?/`~"\`

## 验证文件大小

```bash
# macOS
ls -la 100mb.txt

# Linux
stat 100mb.txt
```
